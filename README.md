# Placement Prediction System

A Machine Learning based web application that predicts whether a student is likely to get placed or not.

## Features
- Placement Prediction
- Probability Analysis
- Personalized Suggestions
- Confusion Matrix
- Feature Importance Graph

## Technologies Used
- Python
- Flask
- Machine Learning
- Random Forest Algorithm
- HTML/CSS/JavaScript
- Scikit-learn

## Input Parameters
- CGPA
- Technical Skills
- Communication Skills
- Internships
- Aptitude Score

## Algorithm Used
Random Forest Classifier
